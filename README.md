<h2 align="center">AMAK Store</h2>

<p align="center">Website using NextJS 14 + TailwindCss + Next Auth + Microservice NET 8 API</p>


## Environment Variables

To run this project, you will need to add the following environment variables to your .env file

`NEXTAUTH_URL`

`NEXT_PUBLIC_API_URL `

`API_URL`

## Run Locally

Clone the project (Node >= 20.9.0)

```bash
  git clone https://github.com/vanthang24803/store-next-14.git
```

Go to the project directory

```bash
  cd store-next-14
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```

Website using port 3000

## Docker

Build Docker Image

```bash
  docker-compose up
```
